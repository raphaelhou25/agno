# 箱体配置表 v2.1

本文档为静态箱体规格目录，箱体设计智能体从此表中检索匹配箱型。  
配置表版本：v2.1 | 更新日期：2025-01 | 总计：32种标准箱型

---

## 字段说明

| 字段 | 说明 |
|------|------|
| `box_id` | 唯一标识符，命名规则：`{箱型}-{瓦楞}-{内L}x{内W}x{内H}` |
| `box_type` | 箱型：RSC（普通开槽箱）/ FOL（全叠盖箱）/ HSC（半开槽箱）/ FRIC（套叠式）|
| `flute_type` | 瓦楞型：A（高强度）/ B（中等）/ C（标准）/ E（微型）/ BC（双瓦楞）/ EB（超强）|
| `inner_L/W/H_mm` | 内尺寸（mm） |
| `outer_L/W/H_mm` | 外尺寸（mm）= 内尺寸 + 2×瓦楞厚度 |
| `max_weight_kg` | 最大装载重量（kg） |
| `bcl_n` | 箱体抗压强度 BCL（牛顿），参考值，实际值受材料影响 |
| `suitable_transport` | 适用运输方式，`all` 表示全适用 |
| `approx_cost_cny` | 参考成本（元/套，基于200g/m²牛皮纸，批量1000+） |
| `notes` | 适用场景备注 |

## 瓦楞型参数速查

| 瓦楞型 | 厚度(mm) | 每面加厚 | 特点 | 适用场景 |
|--------|---------|---------|------|---------|
| E | 1.6 | 2 | 超薄轻量 | 空运、轻小件 |
| B | 3.0 | 3 | 标准通用 | 绝大多数外贸场景 |
| C | 3.5 | 3.5 | 稍强于B | 中等重量，强度要求稍高 |
| A | 4.5 | 4.5 | 高强度 | 重货、高堆叠 |
| BC | 6.0 | 6 | 双瓦楞 | 重货+高压强度要求 |
| EB | 4.6 | 5 | 双瓦楞超薄 | 需要强度但控制重量 |

> **外尺寸计算**：outer = inner + 2 × 每面加厚（L/W方向）；高度加厚略小（约60%），实际见表中数据。

## 箱型速查

| 箱型 | 全称 | 特征 | 优势场景 |
|------|------|------|---------|
| RSC | Regular Slotted Container | 四片开槽，上下盖对叠 | 最通用，成本最低 |
| FOL | Full Overlap Slotted Container | 上下盖完全重叠 | 底部承重强，重物首选 |
| HSC | Half Slotted Container | 无上盖，套盖形式 | 需要频繁开合 |
| FRIC | Friction Lock Box（套叠式）| 内外套接无需胶带 | 瘦高型产品 |
